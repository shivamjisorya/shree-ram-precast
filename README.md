# Shree Ram Precast

Company website for **Shree Ram Precast Concrete Pipes** — manufacturers and suppliers of RCC Hume pipes, manhole systems, SFRC covers, cement paver blocks, concrete road barriers, kerb stones and allied precast products. Based in Rohtak and Jhajjar, Haryana; serving Delhi NCR, Haryana, Uttar Pradesh, Rajasthan, Punjab and Uttarakhand.

---

## Tech stack

- **HTML5** — semantic markup, multiple pages
- **CSS3** — custom styles, responsive layout, animations
- **Vanilla JavaScript** — nav, carousel, testimonials, back-to-top, stats count-up, areas map

No frameworks or build step. Open in a browser or serve with any static host.

---

---

## Running locally

1. Clone the repo and open the folder.
2. Either:
   - **Double‑click** `index.html` to open in your browser, or  
   - **Serve the folder** (e.g. `npx serve .` or your IDE’s “Live Server”) so links and assets load correctly.
